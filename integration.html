Integration formulas

LIST OF BASIC INTEGRAL FORMULAE

The basic integral formulas are given below:

  • ∫ 1 dx = x + C
  • ∫ a dx = ax+ C
  • ∫ xn dx = ((xn+1)/(n+1))+C ; n≠1
  • ∫ sin x dx = – cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ sec2x dx = tan x + C
  • ∫ cosec2x dx = -cot x + C
  • ∫ sec x (tan x) dx = sec x + C
  • ∫ cosec x ( cot x) dx = – cosec x + C
  • ∫ (1/x) dx = ln |x| + C
  • ∫ ex dx = ex+ C
  • ∫ ax dx = (ax/ln a) + C ; a>0, a≠1
  • ∫ tanx.dx =log|secx| + C
  • ∫ cotx.dx = log|sinx| + C
  • ∫ secx.dx = log|secx + tanx| + C
  • ∫ cosecx.dx = log|cosecx - cotx| + C

INTEGRATION FORMULAE OF INVERSE TRIGNOMETRIC FUNCTIONS

  • ∫ 1/(1 +x2).dx = -cot-1x + C

  • ∫ 1/x√(x2- 1).dx = sec-1x + C

  • ∫ 1/x√(x2- 1).dx = -cosec-1x + C

  • ∫1/√(1 - x2).dx = sin-1x + C

  • ∫ /1(1 - x2).dx = -cos-1x + C

  • ∫1/(1 + x2).dx = tan-1x + C
