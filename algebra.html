Algebra formulas

Some Algebra formulas involving squares are:

  • a²– b² = (a – b)(a + b)
  • (a + b)²=  a²+ 2ab + b²
  • a²+ b²= (a + b)²– 2ab
  • (a – b)² = a²– 2ab+ b²
  • (a + b + c)² = a² + b² + c²+ 2ab + 2bc + 2ca
  • (a – b – c)² = a²+ b²+ c²– 2ab + 2bc – 2ca


Some Algebra formulas involving cubes are:
  • (a + b)³ = a³+ 3a²b + 3ab²+ b³
  •  (a + b)³ = a³ + b³ + 3ab(a + b)
  • (a – b)³= a³ – 3a²b + 3ab² – b³
  • (a – b)³= a³ – b³ – 3ab(a – b)
  • a³ – b³ = (a – b)(a²+ ab + b²)
  • a³ + b³ = (a + b)(a²– ab + b²)

  • a⁴ – b⁴= (a – b)(a + b)(a² + b²)
